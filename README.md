# CI/CD Pipeline Project

This project demonstrates a CI/CD pipeline using:

- GitHub Actions
- Tekton Pipelines
- OpenShift

## Features
- Linting with flake8
- Unit testing with nose
- Automated CI/CD workflow

## Project Structure
- app/: Application code
- tests/: Unit tests
- .github/workflows/: CI pipeline
- .tekton/: Tekton tasks
